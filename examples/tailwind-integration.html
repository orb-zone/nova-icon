Tailwind CSS Integration

NovaIcon renders to light DOM, so Tailwind utilities work directly on components

Size Utilities

Use standard Tailwind sizing classes: 

<nova-icon icon="check-icon" class="w-6 h-6"></nova-icon>
<nova-icon icon="check-icon" class="w-8 h-8"></nova-icon>
<nova-icon icon="check-icon" class="w-12 h-12"></nova-icon>

w-6 h-6

w-8 h-8

w-12 h-12

w-16 h-16

w-24 h-24

Color Utilities

Use Tailwind text color utilities: 

<nova-icon icon="star-icon" class="w-10 h-10 text-red-500"></nova-icon>
<nova-icon icon="star-icon" class="w-10 h-10 text-blue-500"></nova-icon>
<nova-icon icon="star-icon" class="w-10 h-10 text-purple-500"></nova-icon>

Hover States

Apply hover effects with Tailwind hover variants: 

<nova-icon 
  icon="heart-icon" 
  class="w-12 h-12 text-gray-400 hover:text-red-500 transition-colors cursor-pointer"
></nova-icon>

Hover over the icons to see the color transition

Transform Effects

Combine transforms with hover states: 

<nova-icon 
  icon="star-icon" 
  class="w-12 h-12 text-yellow-500 hover:scale-125 transition-transform cursor-pointer"
></nova-icon>

hover:scale-125

hover:rotate-90

hover:translate-x-2

hover:rotate-45

Combined Effects

Mix multiple utilities for rich interactions: 

<nova-icon 
  icon="heart-icon" 
  class="w-12 h-12 text-gray-400 hover:text-red-500 hover:scale-110 transition-all duration-300 cursor-pointer"
></nova-icon>
