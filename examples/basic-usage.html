Basic Usage

Learn how to register and render icons with NovaIcon

1. Register an Icon

First, import and register your icons with the NovaIconRegistry: 

import { NovaIconRegistry } from '@orb-zone/nova-icon';

NovaIconRegistry.register('check-icon',
  '<path d="M20 50 L40 70 L80 20" stroke="currentColor" fill="none" />',
  { viewBox: '0 0 100 100' }
);

2. Use in HTML

Add the icon to your markup with the custom element: 

<nova-icon icon="check-icon"></nova-icon>
‚Üê Default size (24px)

3. Set Size
